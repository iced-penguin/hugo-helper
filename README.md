# hugo-helper

hugo-helper helps you build Hugo-generated blog.

## How to use

### Create new article

Execute `hugo-helper new` command.

```
$ hugo-hekper new -h
create new article

Usage:
  hugo-helper new [flags]

Flags:
  -h, --help             help for new
  -s, --section string   section is a directory in which new articles will be placed, under content
```
